## 구현 기능 목록

### 입력
- [ ] 코치 입력 기능을 구현한다.
- [ ] 못 먹는 음식을 입력 받는 기능을 구현한다.

### 출력
- [ ] 출력 메세지들을 enum 으로 구현한다.
- [ ] 메세지 출력 기능을 구현한다.

### 초기값을 저장하는 Configure
- [x] 카테고리 별 메뉴를 초기화하여 저장한다.

### 카테고리 저장소
- [ ] 카테고리 목록을 저장한다.

### 코치 저장소
- [x] 코치 목록을 저장한다.

### 못 먹는 메뉴 저장소
- [ ] 코치를 key, 못 먹는 음식들을 value 로 가지는 저장소를 구현한다.

### 메뉴 추천 기능 구현
- [ ] 월요일에 추천할 카테고리를 무작위로 정한다. 
- [ ] 각 코치가 월요일에 먹을 메뉴를 추천한다.
- [ ] 화, 수, 목, 금요일에 대해 i, ii 과정을 반복한다.
